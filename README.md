# A-QPhone

A-QPhone is a phone/e-commerce management web application developed using ASP.NET Core and Entity Framework Core.

## Features
- Product management
- User authentication
- Shopping cart
- Database management
- Admin dashboard

## Technologies Used
- ASP.NET Core MVC
- C#
- Entity Framework Core
- MySQL
- HTML/CSS
- JavaScript
